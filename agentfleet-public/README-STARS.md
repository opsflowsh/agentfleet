# Adding Stars Chart to README

Add this to the README.md after the title:

```markdown
<div align="center">

![GitHub stars](https://img.shields.io/github/stars/opsflowsh/agentfleet?style=social)
![GitHub forks](https://img.shields.io/github/forks/opsflowsh/agentfleet?style=social)
![GitHub issues](https://img.shields.io/github/issues/opsflowsh/agentfleet)
![License](https://img.shields.io/badge/license-MIT-blue.svg)

[![Star History Chart](https://api.star-history.com/svg?repos=opsflowsh/agentfleet&type=Date)](https://star-history.com/#opsflowsh/agentfleet&Date)

</div>
```

This adds:
- Live star count badge
- Fork count
- Issue count
- Star history chart (shows growth over time)

The chart updates automatically!
